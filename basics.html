Abstract

This is a sweet and simple guide on how to work with LaTex in Overleaf. It covers simple and fundamental commands, formatting, and tips to be as efficient as possible.

Structure

Templates

Overleaf (this is not a feature of LaTex itself) provides you with a variety of templates to use for a variety of different types of projects. Click View All to see even more.

If you're unsure about what type of format you want or prefer to make your own, a "Blank Project" will give you the framework for a very simple document.

Document Formatting, Title, and Table of Contents

The first line defines the type of document; within the preceding brackets font size (in pt) and paper size (a4paper, legalpaper, and letterpaper) can be adjusted as well. The default font size is 10pt and the default paper size is letterpaper (8.5' x 11').

The second line defines the encoding. In general, don't touch this. In the lines after, import the packages you are going to use. If you plan on using images use the "graphicx" package and provide a graphics path if you plan to have the images in sub-directories.

The lines after package imports are used to define the title, author, and date. The "thanks" commands can be used to include names of those who helped in footnotes. The "today" command will update the date to the current date each time the document is compiled.

To start printing words on the document, begin a "document" environment; use the "maketitle" command to print the title, author, and date, and the "tableofcontents" command to create a table of contents which automatically updates each time you recompile.

Additionally, you may make an abstract, which is used to introduce or briefly overview the main topic of the paper.
