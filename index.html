Introduction to LaTeX

LaTeX is a typesetting language most often used for technical orscientific documents, usually of an academic nature.

It takes plain text and creates stylized documents using tags in asimilar way to HTML/CSS.

History

Leslie Lamport developed LaTeX in 1985. LaTeX evolved from TeX, originally developed by Donald Knuth in the late 1970s.

Donald Knuth was working on publishing a book and was frustrated with the typesetting process. He thought that he could just learn to do this himself, and write a program that would produce a betterquality document than what the publishing company was offering.

He estimated the project would take about 6 months, it ended uptaking about 10 years. While TeX gained attention since it worked on any computer and operating system, was free, and would always generate the same document.

TeX and LaTeX

LaTeX heavily relies on TeX macros.

Macros (macroinstructions) are basically small code generators which take a certain input in the code and upon compilation of the code replace it with the preset output, which will later be run. A common example is:

define square(x) ((x) * (x))

n = square(5) which becomes...

n = ((5) * (5))

This allows LaTeX to have complicated tags that allow for things such as bibliographies to be made with only one command.

Common Uses

The most common use is typesetting documents with lots of mathematical equations and code in them. There are also LaTeX add-ins for powerpoints that also make writing equations simpler.

The most basic way to use LaTeX is to download it, write a .tex file and then compile it which would generate the document, there are editors that make the process easier. One such editor is Overleaf.

Pros and Cons

WYSIWYG (what you see is what you get) is the main other form of typesetting. Things like Microsoft Word and Google Docs are WYSIWYG editors.

With LaTeX you don’t immediately see the results. You need to compile it to see the change. Things like Overleaf make this rather easy though.

While LaTeX does provide a number of different output file types (DVI, PDF, HTML, XML), it also has restrictions in how much you can format. If one wants to control every aspect of the layout, have innumerable fonts, images, and other specifics, LaTeX is not the best choice. It is best for clean, professional looking documents, so it does not allow for too much creativity. If you prefer to have a very creative, colorfuland complex layout, LaTeX wouldn’t be able to do that.

LaTeX does have a learning curve, but it takes about half an hour if not less to learn enough to properly use it. It is more efficient and easier for equations and code, and will probably be useful in college.
